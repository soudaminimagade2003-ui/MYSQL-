This project focuses on analyzing **cryptocurrency market data using MySQL**.
It demonstrates how **SQL queries** can be used to extract meaningful insights related to:

* Investors
* Coins
* Exchanges
* Trading volume
* Price trends

## Problem Statement

The cryptocurrency market generates a massive amount of data every day.
However, **raw data alone does not provide value unless it is structured, queried, and analyzed properly**.

### The key challenges addressed in this project are:

* Organizing crypto market data into a **relational database**
* Analyzing **investment patterns**, **coin performance**, and **exchange activity**
* Extracting **actionable insights using SQL only**
